## Overview

This project contains solutions to various JavaScript programming questions covering concepts such as:
- Arrays and array manipulation
- Loops (for, while)
- Functions
- DOM manipulation
- Control structures
- Pattern printing

## Questions

The project is organized into multiple questions covering different JavaScript concepts:

- **Question 1-19**: Individual JavaScript problems
- **Question 20**: Contains three parts (A, B, C) focusing on pattern printing:
  - Part A: Print 4 rows of 5 stars each
  - Part B: Print ascending star pattern (1 to 5 stars)
  - Part C: Print descending star pattern (5 to 1 stars)

Each question is contained in its respective directory with HTML and JavaScript files.

## Installation
